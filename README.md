# sddw

Spec-Driven Development (SDD) is an emerging paradigm that inverts the traditional software workflow: specifications become the source of truth, and code becomes a generated or verified artifact. When combined with agentic AI systems, SDD dramatically improves reliability, auditability, and scalability of AI-generated code. Detailed, executable specs reduce AI code errors by up to 50% (Piskala) and security defects by 73% (Marri). Tests-as-specs achieve 94.3% on SWE-Bench (Han et al.).
